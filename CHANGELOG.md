# Change Log

All notable changes to the "Royal Pine" extension will be documented in this file.

## [1.0.0] - 2024-05-22
### Added
- Initial release of **Royal Pine**.
- Integrated JetBrains Rider Dark UI colors.
- Custom syntax highlighting for Strings (Green), Numbers (Gold), and Variables (Pink).
- Universal support for all programming languages.
- Specific optimizations for PHP/Laravel development.
